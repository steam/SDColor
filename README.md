# SDColor

Dead simple connivence categories for UIColor.

### Installation with CocoaPods

#### Podfile

```ruby
platform :ios, '7.0'
pod "SDColor", "~> 0.1"
```

### Usage

```objective-c
#import "UIColor+SDColor.h"
```

```objective-c
[UIColor hex:0xFF00FF];
```

```objective-c
[UIColor hex:0xFFFF00 alpha:0.5];
```
